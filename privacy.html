Privacy Policy

Effective Date: 2025-08-22

1. Introduction

This privacy policy explains how the Wplace Tracer Pro Extension handles user data. This extension is designed exclusively for Wplace.live to allow users to upload template images and overlay them on the canvas for precise pixel art creation and tracing.

2. Data Collection

This extension does not collect any personally identifiable information or sensitive user data. The only data accessed is the content on https://wplace.live/* necessary to provide the overlay functionality, specifically user-uploaded template images and the canvas content while the extension is active.

3. Types of Data Not Collected

  • Personally identifiable information (name, email, address, etc.)
  • Health information
  • Financial or payment information
  • Authentication information (passwords, credentials, PINs)
  • Personal communications (emails, chats, texts)
  • Location information (GPS, IP, etc.)
  • Web history or other browsing activity
  • Other user activity beyond the overlay functionality

4. Remote Code

This extension does not use remote code. All scripts and functionality are included within the extension package. No external scripts or code are loaded at runtime.
